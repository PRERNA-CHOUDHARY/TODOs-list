# TODOs-list
ToDo list is a simple web application to Note down Your daily tasks in order not to miss anything. 📝

### Features 

* Add new task 📆
* Mark task as completed ✔
* Edit task ✏
* Delete single task 🚫
* Delete completed tasks

### Languages/frame-work used:

* HTML  
* CSS
* JavaScript
* BootStrap
